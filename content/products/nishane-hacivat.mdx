## Full description

Hacivat opens with a sharp, bright **pineapple + grapefruit + bergamot** freshness, then settles into **cedar and patchouli** with an **oakmoss-woody** base that feels crisp, confident, and expensive.

## How to use

1. Apply to neck and wrists (start light).
2. Let it dry down before adding more.
3. Reapply only if needed; this profile carries.

**Pro tip:** Perfect for clean outfits and "sharp" occasions.
